# Support

Use GitHub Discussions for setup questions, install help, and receiver stack
notes:

https://github.com/Ident-1090/Ident/discussions

Use GitHub Issues for:

- reproducible bugs
- receiver compatibility reports
- UI layout problems
- documentation gaps
- feature requests tied to a receiver workflow

Use GitHub private vulnerability reporting for security issues:

https://github.com/Ident-1090/Ident/security/advisories/new

## Before Asking for Help

Please collect:

- Ident version
- receiver stack and operating system
- install method
- path where `aircraft.json` is written
- browser and device
- relevant `identd` logs

Do not post feeder keys, exact receiver coordinates, private hostnames, tunnel
URLs, or screenshots that reveal private location details.
